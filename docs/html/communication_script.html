Mission Planner Communication Script Creates a communication link between the Backend Server and Mission Planner. Utilises MavLink and is run as a Mission Planner Script.

Script Usage From Mission Planner Home, navigate to “Simulation” -> “Plane” -> “Stable”. On the left side, navigate to “Scripts” -> “Select Script” -> “Upload” ->

-> “Run Script”
